- [ ] **Cross-projection**: D-in-G and G-in-D fractions at k=10, 20
- [ ] **Scaling analysis**: Plot eff-dim vs model hidden dimension
- [ ] Reuse: `negative_subspace_geometry.py` (adapt for multi-model)
- Output: `_Results/exp2_subspace_characterization.md`

### Exp-5: Toy Model (CPU, parallel with Exp-1)

- [ ] **Implement linear associative memory**: W = sum v_i k_i^T
- [ ] **Parameter sweep**: d={100,200,500,1000}, n={10,20,50,100}, 10 seeds each
- [ ] **ROME-style editing**: (C+lambda*I)^{-1} k* (v_new - Wk*)^T
- [ ] **Gradient attribution**: g_i = (Wk_i - v_i) k_i^T, g_M = mean/norm
- [ ] **Compute**: TECS, subspace analysis, principal angles, C^{-1} ablation
- [ ] **Ground truth**: angle between delta_W and true knowledge direction
- [ ] **Output**: 2D heatmap of Cohen's d vs (d, n). Phase boundary curve.
- Output: `_Results/exp5_toy_model.md`
