## Phase 2a Full: BSS Cross-Seed Stability (~7 GPU-hours) — CRITICAL GATE

### Pre-requisites
- 5 trained ResNet-18 models (seeds 42, 123, 456, 789, 1024)
- 500 test points (50/class, stratified) — same indices used in Phase 1

### Exp-2a.1: Multi-Seed Model Training (~1h)

Script: `Codes/experiments/phase2a_bss_crossseed.py` (training section)

- [ ] Train ResNet-18 seeds 789, 1024 (if not already available)
  - 200 epochs, CIFAR-10, same hyperparameters as seed 42
  - Save checkpoints to `Codes/_Data/models/resnet18_seed{789,1024}.pt`
- [ ] Verify all 5 models achieve 94-96% test accuracy
- [ ] Record test accuracy per seed

### Exp-2a.2: K-FAC Factor Computation (5 seeds x ~20min = ~1.5h)

Script: `Codes/experiments/phase2a_bss_crossseed.py` (factor section)

- [ ] Per seed: Compute K-FAC factors from 5000 training samples (full model)
- [ ] Per seed: Kronecker eigendecomposition, top-100 eigenvalues + eigenvectors
- [ ] Save: eigenvalues, eigenvectors, K-FAC factors to `Codes/_Data/phase2a/seed_{s}/`
- [ ] Report: eigenvalue spectrum summary per seed (max, min, distribution shape)
- [ ] **Verify**: eigenvalue scale consistent across seeds (~5e-05 max, as in pilot)

### Exp-2a.3: BSS Computation (5 seeds x 500 pts x ~5min = ~4h)

Script: `Codes/experiments/phase2a_bss_crossseed.py` (BSS section)

- [ ] Per seed, per test point: compute BSS_raw, BSS_partial, BSS_ratio
- [ ] **Adaptive bucket thresholds** (percentile-based per seed):
  - Outlier: top 0.2% of eigenvalues
  - Edge: next 0.5%
  - Bulk: remaining 99.3%
- [ ] **BSS_partial**: regress out ||g_z||^2 per bucket, per seed
- [ ] **BSS_ratio**: BSS_outlier / BSS_total per test point per seed
- [ ] Save: BSS arrays (500 x 3 variants x 3 buckets) per seed to `Codes/_Data/phase2a/`
- [ ] Report: descriptive statistics per seed per variant

### Exp-2a.4: Cross-Seed Stability Analysis

Script: `Codes/experiments/phase2a_bss_crossseed.py` (analysis section)

- [ ] **Spearman rho matrix**: 10 seed pairs x 3 BSS variants = 30 correlations
- [ ] **Mean pairwise rho** per variant (primary stability metric)
- [ ] **ICC(2,1)** per variant (reliability metric, Empiricist recommendation)
- [ ] **Baselga decomposition**: J10 replacement vs reordering per seed pair
- [ ] Save: correlation matrices, ICC values

### Exp-2a.5: BSS Predictive Power

- [ ] Per seed: Spearman(BSS_outlier, per-point LDS)
- [ ] Per seed: Partial Spearman(BSS_outlier, LDS | class + ||g||)
- [ ] Cross-seed average of partial correlations

### GATES (Phase 2a)

| Criterion | Pass | Borderline | Fail | Action on Fail |
|-----------|------|-----------|------|----------------|
| BSS_partial cross-seed rho | > 0.5 | 0.3-0.5 | < 0.3 | **iterate_direction** |
| Within-class BSS variance | > 25% | 15-25% | < 15% | BSS is class proxy, downgrade |
| Partial corr (BSS vs LDS \| class+grad) | > 0.15 | 0.10-0.15 | < 0.10 | BSS adds nothing beyond grad norm |

**Kill-switch**: BSS_partial cross-seed rho < 0.3 → immediate iterate_direction. Do NOT proceed to Phase 3.

## Phase 2a Augmented: Controls & Diagnostics (~1 GPU-hour)

Run concurrently with or immediately after Phase 2a full.

### Exp-2a.6: Randomized-Bucket Mechanism Control (0 GPU-hours)

Script: `Codes/experiments/phase2a_randomized_bucket.py`

**BINDING CONDITION from design_review.**

- [ ] Using cached eigenvalues + gradients from Exp-2a.2/2a.3:
  - Compute BSS with **random** bucket assignments (1000 permutations)
  - Random assignment: shuffle eigenvalue-to-bucket mapping, keep bucket sizes fixed
- [ ] Compare: Spearman(BSS_random, LDS) vs Spearman(BSS_magnitude, LDS)
- [ ] **Interpretation**:
  - If random BSS has >80% of magnitude-based BSS predictive power → bucket structure is irrelevant, BSS = gradient-subspace projection (acknowledge in paper)
  - If random BSS has <50% of magnitude-based BSS predictive power → bucket structure carries signal
- [ ] Report: distribution of random BSS correlations with p-value for magnitude-based

### Exp-2a.7: Per-Seed ANOVA Cross-Validation (0 GPU-hours)

Script: `Codes/experiments/phase2a_anova_crossseed.py`

**BINDING CONDITION from design_review.**

- [ ] Per seed (5 seeds): Two-way ANOVA on J10 and LDS
  - Recompute J10 and LDS per seed using that seed's attributions
  - Report per-seed residual R^2 for both metrics
- [ ] **Stability check**:
  - If J10 residual drops below 50% on any seed → qualify J10 claim
  - If LDS residual drops below 30% on any seed → qualify LDS claim
- [ ] Report: 5-seed mean and std of residual R^2 for each metric

### Exp-2a.8: Principal Angle Diagnostic (0 GPU-hours)

Script: `Codes/experiments/phase2a_principal_angles.py`

- [ ] Compute principal angles between top-K eigensubspaces across 5 seeds (10 pairs)
  - K = {10, 20, 50} (outlier subspace sizes)
- [ ] Compare with random-subspace null (1000 random K-dim subspaces)
- [ ] **Purpose**: validate that BSS_ratio's outlier subspace is stable across seeds
- [ ] Report: mean principal angle per K, p-value vs random null

### Exp-2a.9: Early Damping Ablation (~1 GPU-hour)

Script: `Codes/experiments/phase2a_damping_ablation.py`

- [ ] Test BSS at delta = {1e-06, 1e-04, 0.01, 0.1, 1.0}
  - Use seed 42, 500 test points
  - Recompute BSS_raw per damping level (perturbation factors change)
- [ ] Report: BSS-LDS correlation at each damping level
- [ ] **Key question**: Does BSS behavior change at delta ~ lambda_max (~5e-05)?
  - If invariant to damping → confirms gradient-geometry interpretation
  - If behavior changes at delta ~ lambda_max → richer theoretical story possible

### Kill-Switch Checkpoints (from Formalize Review)

These are 0-cost checks using existing data. Run BEFORE committing to Phase 2a GPU work:

- [ ] **Perturbation factor ratio** (outlier/bulk): compute from pilot eigenvalues

  - Already observed in pilot (~90 across all buckets), just formalize the check
- [ ] **Eigenvalue rank stability across seeds**: using seeds 42, 123, 456 from probe

  - If rho > 0.8 → eigenvalue spectrum is stable (supports BSS rationale)
- [ ] **Partial BSS cross-seed rho from 3-seed probe data**:
  - Quick estimate from existing 3 seeds x 100 points
  - Gate: if rho < 0.3 from 3-seed estimate → iterate_direction BEFORE training seeds 789/1024
